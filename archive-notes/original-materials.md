# Original Local Materials Notes

The original local folder contained project files mixed with personal, third-party, and oversized materials. This GitHub-ready repository keeps the project-specific materials and excludes files that are unsuitable for a public academic portfolio.

Included:

- MaixPy/K210 project source code.
- K210 `.kmodel` file and training report.
- Sample image dataset and XML annotations.
- Project proposal, report, presentation slides, and demo videos.

Excluded from this public repository:

- Personal resume files.
- Unrelated software folders such as Steam community tools.
- Third-party K210/MicroPython development kits and tutorial packages.
- Executables, DLLs, registry files, drivers, and installers.
- Oversized compressed archives such as the full local backup zip files.
- Duplicate copies of materials already included in cleaner form.

The excluded materials can remain in the local archive folder and should only be published separately if licensing, privacy, and GitHub file-size constraints are checked.
